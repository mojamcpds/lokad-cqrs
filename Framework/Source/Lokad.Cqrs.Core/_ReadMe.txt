Lokad.CQRS.Core actually implements interfaces declared in Lokad.CQRS.Shared. It includes:

* Configuration Builders;
* Message/Consumption mapping;
* Lokad Message Format;
* Basic support for the Azure Views;
* Scheduled Tasks;
* Generic queue management;

See http://code.google.com/p/lokad-cqrs/ for more details on these features.
